# Brazilian E-Commerce Marketplace Analytics

### End-to-End Data Analytics Project

Python | SQL | MySQL | Power BI | DAX | ETL

---

## Project Overview

This project analyzes the Brazilian Olist E-Commerce Marketplace dataset to understand business performance across sales, customers, sellers, and logistics.

The project demonstrates a complete data analytics workflow, starting from raw data cleaning and transformation, followed by SQL-based business analysis, and ending with interactive Power BI dashboards for business decision-making.

The goal of this project is to convert raw e-commerce data into meaningful business insights that help stakeholders monitor performance and identify opportunities for improvement.

---

## Business Problem

E-commerce businesses generate large volumes of transactional data every day. Without proper analysis, it becomes difficult to answer important business questions such as:

- Is revenue increasing over time?
- Which product categories generate the highest sales?
- Which sellers contribute the most revenue?
- How satisfied are customers?
- Are deliveries happening on time?
- Which business areas require improvement?

This project answers these questions through SQL analysis and interactive Power BI dashboards.

---

## Project Objectives

The objectives of this project are:

- Clean and prepare raw e-commerce data using Python.
- Perform Exploratory Data Analysis (EDA).
- Design and load a relational database in MySQL.
- Create SQL reporting views for business analysis.
- Develop interactive Power BI dashboards.
- Build business KPIs using DAX.
- Present business insights in a clear and interactive format.

## Dataset Information

**Dataset Name**

Brazilian Olist E-Commerce Dataset

The dataset contains information about customers, orders, products, sellers, payments, reviews, and deliveries from a Brazilian online marketplace.

### Main Tables

| Table | Description |
|-------|-------------|
| Customers | Customer information |
| Orders | Order details |
| Order Items | Products sold in each order |
| Products | Product information |
| Sellers | Seller information |
| Payments | Payment details |
| Reviews | Customer reviews |
| Geolocation | Customer and seller locations |
| Product Category Translation | Portuguese to English category names |

## Dashboard Overview

This project contains five interactive dashboards and four detailed reports.

### Executive Dashboard

Provides an overall view of business performance using key business metrics such as revenue, orders, customers, average order value, customer ratings, and on-time delivery.

### Sales Dashboard

Analyzes sales performance, monthly revenue trends, order trends, freight costs, product category performance, and revenue distribution.

### Customer Dashboard

Provides insights into customer growth, customer distribution, customer ratings, purchasing behavior, and customer satisfaction.

### Operations Dashboard

Monitors delivery performance, logistics efficiency, freight costs, delivery delays, and on-time delivery percentage.

### Seller Dashboard

Analyzes seller performance using revenue, orders, customer ratings, delivery performance, and seller performance tiers.
